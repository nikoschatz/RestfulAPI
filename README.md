# Java Rest API

### A Java application exposin a JSON Rest API.


<br>

#### In order to use this code you will need:

**Instructions**<br>

- Eclipse IDE
- Java 13
- Spring Boot 2.2
- Maven
- Java Persistence API (JPA)
- H2 Database


#### You will also need these dependencies

**Dependencies**<br>
-Spring Data JPA
-Rest Repositories
-Rest Repositories HAL
-H2 Database
